# SafeCam Lite Pro Night

旧安卓手机监控增强版，新增夜间低光增强功能。适合自有店铺、仓库、宠物、院子等明确授权场景使用。

## 重要说明：这不是真红外夜视

普通手机摄像头没有真正的红外夜视能力，完全黑暗环境下无法凭空看清。
本版本实现的是“低光增强”：

- 数字提亮
- 对比增强
- 黑白增强
- 可选打开手机手电筒补光

如果需要真正全黑环境夜视，需要外接红外补光灯/低照度摄像头/专用监控摄像机。

## 功能

- 实时网页观看
- 循环记录
- 网页回放
- 自动删除旧记录
- 设置记录间隔
- 设置最多保留小时数
- 设置最大占用空间
- 夜间低光增强
- 手电筒补光
- 黑白增强模式
- 开机自启开关
- 外出观看方案：Tailscale / ZeroTier 虚拟局域网

## 推荐夜间参数

OPPO Reno3 Pro 5G / Android 12 可以先用：

- 夜间低光增强：开启
- 手电补光：按场景开启
- 黑白增强：开启
- 夜间提亮：45
- 夜间对比：25
- 记录间隔：3-10 秒
- 最大空间：1024-4096 MB

## 使用方式

1. 旧手机安装 APK
2. 打开 App，允许摄像头和通知权限
3. 设置 PIN、循环记录、夜间增强
4. 点 `START CAMERA SERVER`
5. 同 Wi-Fi 下另一台手机打开实时地址
6. 回放打开 `/recordings?pin=你的PIN`

## 长期运行建议

- 插电
- 关闭省电限制
- 允许后台运行
- 保持通风
- 如果开启手电补光，注意温度和耗电

## 外出实时观看

不建议路由器端口映射到公网。
推荐 Tailscale / ZeroTier：

1. 旧手机安装 Tailscale 或 ZeroTier
2. 外出查看的手机也安装同一个软件
3. 两台手机加入同一个虚拟网络
4. 打开本 App，如果地址里出现 100.x.x.x 之类虚拟 IP，就用这个地址访问
5. 示例：`http://100.x.x.x:8080/?pin=123456`

## 安全说明

本项目不隐藏运行、不静默偷拍、不录音、不绕过系统权限。
运行时有前台通知，请只用于你拥有或已获得授权的场所。
