# SafeCam Lite

这是一个适合旧安卓手机的简易“自有设备监控端”源码包。

## 功能
- 旧安卓手机作为摄像头端
- 另一台手机/电脑用浏览器查看
- 本地 HTTP 访问，带简单 PIN
- 前台服务常驻通知，不做隐藏运行
- 默认 640×480、约 2 FPS，适合旧手机低发热长期运行

## 使用方法
1. 用 Android Studio 打开本文件夹。
2. 等待 Gradle 同步。
3. 连接旧安卓手机，点击 Run 安装。
4. 打开 App，允许摄像头权限和通知权限。
5. 修改 PIN，点击 Start Camera Server。
6. 在另一台手机浏览器打开 App 显示的地址，例如：
   `http://192.168.1.23:8080/?pin=123456`

## 外网查看
不建议把 8080 端口直接暴露到公网。
推荐：
- 两台手机都安装 Tailscale 或 ZeroTier
- 加入同一个虚拟网络
- 查看端打开监控手机的虚拟 IP + 端口

## 长期运行建议
- 插电使用
- 关闭系统省电限制
- 关闭自动清理后台
- 保持 Wi-Fi 稳定
- 降低分辨率和帧率，减少发热
- 手机摆放处贴明示标识，避免侵犯他人隐私

## 不包含
- 不做隐藏图标
- 不做静默偷拍/偷录音
- 不绕过权限
- 不自动开机偷拍
- 不上传第三方服务器

## 手机云端打包 APK
如果你没有电脑或 Android Studio，可以使用本项目内置的 GitHub Actions 自动打包。
查看文件：`手机_GitHub云端打包教程.md`

简单流程：
1. 手机把整个项目上传到 GitHub 仓库
2. 打开 Actions
3. 运行 Build SafeCam Lite APK
4. 在 Artifacts 下载生成的 APK
